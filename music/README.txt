Drop your music here and list the filenames in playlist.json:

  "menu" — starts on a random track, cycles to the next when a song ends
  "race" — same rotation; either loops if you only list one track

menuCooldown / raceCooldown set the seconds of quiet between songs.

Any format the browser plays works (mp3, ogg, m4a...). Missing files are
skipped silently. Volume is mixed in-game; M mutes everything.
